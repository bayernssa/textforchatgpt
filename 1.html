Dieser Button hat den Namen hungerButton. Wenn der Button hungerButton gedrückt wird, soll der Wert 1 in einem Variable hunger gespeichert werden. Dadrunter ist ein Button mit der Beschriftung "Tag erfolgreich!". Dieser Button hat den Namen tagButton. Wenn der Button tagButton gedrückt wird, soll der Wert 1 in einer Variable tag gespeichert werden. Dadrunter ist ein Button mit der Beschriftung "Sport gemacht!". Dieser Button hat den Namen sportButton. Wenn der Button sportButton gedrückt wird, soll der Wert 1 in einer Variable sport gespeichert werden. Dadrunter kommt ein Button mit der Beschriftung „Tagesverbrauch“. Wenn der Button gedrückt wird, öffnet sich ein Fenster oder ein Dialog mit folgenden Eigenschaften: Es gibt ganze oben einen Text "kcal" und rechts daneben ein Eingabefeld. Das Eingabefeld hat den Namen kcalFeld. Dadrunter kommt ein Text "KHs" und rechts daneben ein Eingabefeld. Das Eingabefeld hat den Namen khFeld. Dadrunter kommt ein Text "Fette" und rechts daneben ein Eingabefeld. Das Eingabefeld hat den Namen fetteFeld. Dadrunter kommt ein Text "Eiweiß" und rechts daneben ein Eingabefeld. Das Eingabefeld hat den Namen eiweissFeld. Ganz unten kommt ein Button mit der Beschriftung "Save" hin. Dieser Button hat den Namen saveButton. Wenn der Button saveButton gedrückt wird, soll der Wert aus kcalFeld in einer Variable kcal, der Wert aus khFeld in einer Variable kh, der Wert aus fetteFeld in einer Variable fette und der Wert aus eiweissFeld in einer Variable eiweiss gespeichert werden und der Dialog schließt sich und kehrt zur ursprünglichen Ansicht zurück. In dem Dialog ist auch ein Button Abbrechen. Wenn dieser gedrückt wird, schließt sich auch das Dialogfeld. Generiere Beispielcode für die die Overfläche und eine MainActivity.java.
